# cvDrawTest 

## Dependencies
mediapipe
opencv
cvzone

## How to run
1. Create Python venv in project folder
- python3 -m venv your-venv-name
2. Go into venv 
- source your-venv-name/bin/activate
3. Install dependencies
- pip install mediapipe
- pip install opencv
- pip install cvzone
4. Run main.py
- python3 main.py
